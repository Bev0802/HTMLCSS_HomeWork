Каталог

Отвертка-тестер

является самым простым инструментом для определения наличия напряжения в любой электрической сети. Внешне инструмент напоминает обычную отвертку, но имеет световой индикатор.

Пассатижи

многофункциональный инструмент, по назначению похожий на плоскогубцы. Используется для слесарных работ или манипуляций с электросетями. Отличие заключается в том, что на рабочей области с насечками предусмотрена небольшая выемка, используемая для захвата и фиксации гладких цилиндрических деталей.

Разводной ключ

гаечный ключ переменного размера, используемый для вращения гаек, болтов и других деталей. Является разновидностью рожкового ключа с регулируемыми губками.
